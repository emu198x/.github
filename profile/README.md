# Emu198x

Cycle-accurate, cross-platform emulators for the retro machines of the 198x era,
written in Rust.

Many of the classic systems are well served on Windows but thin on the ground
for macOS and Linux. Emu198x treats accuracy as a craft commitment — validated
against the standard test suites — and cross-platform availability as a
first-class goal, filling those gaps system by system.

It's a long-haul project, built in the open.

## Repositories

- **[emu198x](https://github.com/emu198x/emu198x)** — the emulator workspace: a
  Rust workspace of CPU cores, chip emulations, and machine wirings, with a
  scriptable/MCP surface for inspection and testing.

## Part of the 198x family

A sibling to retro-computing curriculum, assembler, and asset-tooling work that
share a hardware-reference layer — [Code198x](https://github.com/code198x)
(curriculum), [Asm198x](https://github.com/asm198x) (assemblers), and
[Cat198x](https://github.com/cat198x) (asset cataloguing). Emulators, teaching,
and tooling: different views of the same machines.
