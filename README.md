# .github

Org-level GitHub configuration for [Emu198x](https://github.com/emu198x).

- [`profile/README.md`](profile/README.md) — the org profile page shown at
  <https://github.com/emu198x>.
- Community-health files (CODE_OF_CONDUCT, CONTRIBUTING, SECURITY, SUPPORT) and
  the issue/PR templates here act as org-wide defaults: GitHub falls back to
  them for any repo in the org that doesn't ship its own.
